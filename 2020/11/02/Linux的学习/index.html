ls&ll的使用

ls      |可以显示当前文件夹中的所有文件

    -a                    |   列举目录中的全部文件，包括隐藏文件

    -l                    |   列举目录中的细节，包括权限、所有者、组群、大小、创建日期、文件是否是链接等

    -f                    |   列举的文件显示文件类型

    -r                    |   逆向，从后向前地列举目录中内容

    -R                    |   递归，该选项递归地列举当前目录下所有子目录内的内容

    -s                    |   大小，按文件大小排序

    -h                    |   以人类可读的方式显示文件的大小，如用K、M、G作单位

    ls -l examples.doc    |   列举文examples.doc的所有信息

ll      |可以显示当前文件夹中所有文件的所有信息（文件的读取权限，用户组，大小，日期）

ll命令详解

cat命令:

cat命令可以用来合并文件，也可以用来在屏幕上显示整个文件的内容。

cat snow.txt 该命令显示文件snow.txt的内容，ctrl+D退出cat。

pwd&cd的使用

pwd     |显示出当前文件夹一个完整的绝对路径

cd      |进入我们想去的文件夹 

    cd /      转到根目录中 
    cd ~     转到/home/user用户目录下 
    cd /usr 转到根目录下的usr目录中-------------绝对路径 
    cd test 转到当前目录下的test子目录中-------相对路径

相对路径 |没有/开头的路径

绝对路径 |以/开头的路径

Tab键   |减少键盘输入

cd ..   |退回上级目录（cd ../..多次退回）

cd /    |退回根目录

mv（move）：移动或重命名

-i | 互动：如果选择的文件会覆盖目标中的文件，他会提示确认

-f | 强制：它会超越互动模式，不提示地移动文件，属于很危险的选项

-v | 详细：显示文件的移动进度

mv [参数1: 源文件或目录][参数2:目标文件或目录]

1|当参数2是文件时，该命令会将源文件或目录移动到目标文件指定目录并且重命名成目标文件

2|当参数2是目录时，该命令将源文件或目录移动到目标目录

cp(copy) :复制

-i | 互动：如果文件将覆盖目标中的文件，他会提示确认

-r | 递归：这个选项会复制整个目录树、子目录以及其他

-v | 详细：显示文件的复制进度

cp [选项: -R] [参数1:源文件或目录][参数2:目标文件或目录]

4种情形:

1|复制源文件到指定目录,目标文件不存在时， 会生成新文件

2|复制源文件到指定目录,目标文件存在时，系统会询问是否覆 盖已存在文件

3|复制源目录到指定目录，目标目录已经存在时，源目录会复制到目标目录里

4|复制源目录到指定目录，目标目录不存在时，直接以目标目录名生成新目录

scp(security copy) :不同服务器间复制

scp[-r] [-P端口数] [参数1:源文件或目录] [参数2: 目标文件或目录]

情形:

1|从本地复制文件到远程

scp -P 22 localfile root@gongchengketang.com:/home/lizhenrong/

2|从远程复制文件到本地

scp -P 22 root@gongchengketang.com:/home/lizhenrong/remotefile .

3|上面两种情形是文件夹时，添加-r

scp-r -P 22 root@gongchengketang.com:/home/lizhenrong/remotedir .

touch&mkdir

touch|创建一个单一文件

mkdir|创建一个新的文件夹

rm(remove) :删除文件

rm[-r] [-f] [源文件或目 录]

情形:

1|删除文件,系统给提示 rm testfile

2|强制删除文件，系统不给提示rm -f testfile

3|删除文件夹，系统给提示rm -r testdir

4|删除文件夹，系统不给提示rm - rf testdir

5|删除当前目录所有文件(慎用)rm* -rf

top:查看进程情况

top显示出得进程信息默认是按CPU的使用情况来排序的，不过有以下参数可以使用

M (大写) :按内存使用情况排序

P :按CPU使用情况排序

T:按累计时间排序

chown:修改文件的拥有者和组

chown[选项][所有者] :[组]文件 .

管理员账号用来管理文件的拥有者的命令

chmod: 修改文件的权限

chmod [选项][权限数]文件

-   ---     ---       ---
d   uesr    group     other   的权限

user、group、other分别用一个数字表示

例子1： chmod 751 file ， 755=rwx r-x r-x

    r=4|读权限  w=2|写权限  x=1|可执行权限

例子2： chmod a=w  等同 chmod ug=w,o=w 等同  chmod 222

例子3： chmod +x 或则 chmod -x  |加或减权限

tail：查看文本末尾信息

tail [选项]  文件

常用：

tail -n(num)  查看末尾 num 行文本    |查看第**行的文本

tail -(num)f  持续查看末尾num 行文本  |查看日志文件从**行开始的文本

wget：下载文件

wget [选项] [ur]  |

常用方式:

1.直接下载wget http://36kr.com/

2.下载以自定义文件名保存wget -O temp(文件名) http://36kr.com/

3.断点下载wget -c http://36kr .com/

ping：判断某个服务器/地址是否还在运行

ping --help

vim：文本编辑器&文本编辑命令

vim/vi [选项]  文件
编辑模式:

1. 一般模式 ： 可以复制，删除文本，但是不能插入

2. 插入模式 ： 在一般模式下按“I,l,o,0,a,A,r,R” 进入到插入模式，可以任意编辑

3. 命令行模式: 一般模式下按 “：”，进入命令模式

    命令模式的常用命令: 

        :q退出      :q!强制退出     :W保存     :wq(:x)保存后退出     :wq!(:x!)强制保存后退出

grep命令：grep命令的最大功能是在一堆文件中查找一个特定的字符串

grep money test.txt 
以上命令在test.txt中查找money这个字符串，grep查找是区分大小写的。

echo命令：

echo命令是我们常用的一个命令，它的主要作用功能是在屏幕上显示文字，也可以直接在文件中写入要写的内容，具体如下。

echo命令我们经常使用到的选项有两个，一个是-n，表示输出之后不换行。另一个是-e，表示对于转义字符按对应的方式进行处理，假设不加-e，那么在输出时转义字符会按照普通字符进行处理，并不会达到自己想要达到的目的。

rmdir命令:rmdir命令用来删除目录。

mount命令:

mount命令的功能是挂载文件系统，可以挂载硬盘、光盘、软盘，也可以挂载NFS网络文件系统。这个命令的标准用法如下：

mount –t 设备类型 存放目录

mount IP地址：/所提供的目录 存放目录

(无)  不加任何参数，直接输入命令可以显示已挂载的文件系统和目录

-a  挂上/etc/fstab下的全部文件系统

-t  制定所挂上来的文件系统的名称，所有系统支持的文件系统，这个信息可以在/proc/filesystems这个文件里看到

-n  挂上文件系统，但是不把文件系统的数据写入/etc/mtlab这个文件

-w  将文件系统设为可读写

-r  挂上来的文件系统设为只读

在目录/mnt下，挂上iso9660文件系统。输入命令：

mount –t iso9660 /dev/hdb /cdrom
